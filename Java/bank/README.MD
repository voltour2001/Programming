
Compile & Run Command:

    Current Directory:
        javac *.java & java -cp .. bank/Main

    Parent Directory:
        javac bank/*.java & java bank/Main



ABOUT:

Simple program that allows users to register an account and then able to log in, transfer and receive funds and also check log history.
